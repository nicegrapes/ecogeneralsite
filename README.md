### Quick Start
Unzip the `alpha-theme-release.zip` file into any directory. Copy your theme of choice and within that folder run the below commeands from your cli:

```bash
cd path/to/project
yarn
yarn serve

# OR

cd path/to/project
npm install
npm run serve
```
